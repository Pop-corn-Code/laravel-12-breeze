# Laravel 12 + Breeze + Custom Template | Admin Starter Kit

A complete Laravel 12 starter project integrated with Breeze for authentication, customized with a modern template for admin backend. This project demonstrates how to install Breeze in Laravel 12, replace the default authentication pages with a custom design, and build a fully functional system with admin (backend) interfaces.
